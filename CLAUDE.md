## 프로젝트 한 줄 요약

삼국지 배경의 **토탈워류 전략 게임**. 웹 기반(모바일 우선 → PC), **1인 소규모 프로토타입**.
현재 단계: **기획 문서 작성** (구현 전).

## 핵심 설계 (확정된 것)

- **3개 레이어 + 연속 줌**
  - **경영**: 턴제, 전체 지도. 내정·외교·부대 편성·이동.
  - **전략**: 실시간, **Hex 전장**. 부대 단위 지휘. **교전은 자동 계산**(능력치·병종상성·전방·지형·장수·협공).
  - **전술**: 실시간, 특정 교전을 **확대**해 유저가 직접 개입. 장수·개별 병종·지형 연출이 보임.
- **핵심 메커니즘**
  - **액티브 포즈**: 유저가 개입(명령)하는 순간 전체 정지. 판단력 > 반응속도.
  - **이중 시계**: 전략은 빠르게, 전술은 느리게 흐름(`TIME_RATIO`). 둘은 공유 X, 비율로 분리.
  - **주의의 안개**: 전술 몰입 중 벌어진 전략 상황을 알림으로 공유(예: "A부대 후퇴 중").
- **유닛**: Hex 6면 + **전방(facing)**, 한 타일 한 유닛. 병종 삼각상성(창>기>궁>창). **사기 붕괴**로 결판.
- **기술 방향(미확정)**: PixiJS + TypeScript + Vite, 클라이언트 온리. 줌인 교전 1건만 정밀 시뮬.

## 작업 규칙 (convention)

- 문서는 **living document** — 결정이 바뀌면 관련 `doc/*`와 이 파일을 함께 갱신.
- 밸런싱 수치는 **코드가 아니라 데이터로** 둔다(JSON/TS 분리).
- 커밋 후 `git push`로 동기화(이게 PC 간 작업 이어가는 통로).
- 문서는 한국어로 작성.

## 다른 PC에서 이어받는 법

1. `git clone https://github.com/goalgle/santoku.git && cd santoku`
2. 그 폴더에서 `claude` 실행 → 이 `CLAUDE.md`와 `doc/`를 읽고 맥락 복원.
3. (선택) 이전 대화 흐름까지 그대로 옮기고 싶으면, 기존 PC의
   `~/.claude/projects/<프로젝트경로>/<세션id>.jsonl`을 새 PC의 동일 경로에 복사 →
   `claude --resume`로 선택. ⚠️ 비공식 방법이며, 절대경로가 같아야 하고 버전에 따라 깨질 수 있음.
   → **권장은 이 파일 기반으로 새 세션을 시작하는 것.**
