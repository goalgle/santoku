# 07. 엔티티 데이터 설계 (장수 · 병종 · 지형) — 🟡 OPEN

> **상태: 미확정 (작업 중).** 장수·병사·지형의 능력치와 특징, 페널티를 정리해 나가는 문서.
> 수치는 모두 **제안/예시**이며, 프로토타입 구현 후 밸런싱하며 확정한다.

## 7.1 장수 (Characters / Generals) — 🟡 OPEN

삼국지의 핵심 재미. 부대를 지휘하거나 도시를 다스린다. 전술 확대 시 **본인이 등장**한다.

### 능력치 (초안 — 코에이류 4스탯 기반)
| 능력치 | 영향 (제안) | 사용 레이어 |
|--------|-------------|-------------|
| 무력 (武) | 근접 전투력, 일기토, 돌격 위력 | 전략·전술 |
| 지력 (智) | 계략·매복·화공, 함정 간파 | 전략·전술 |
| 통솔 (統) | 부대 사기·방어·이동 통제, 협공 저항 | 전략 |
| 정치 (政) | 내정 생산·민심·건설 효율 | 경영 |

### 정할 것 (OPEN)
- [ ] 능력치 4종으로 충분한가? (매력/속도 등 추가 여부)
- [ ] **장수의 전투 역할**: 부대 버프형인가 / **일기토** 같은 개별 전투 주체인가? (→ [04 4.7 #5](./04-tactics-layer.md))
- [ ] **특수기(스킬)** 체계: 능력치 파생 vs 개별 보유, 쿨다운/발동 조건
- [ ] 장수 사망/부상 규칙 (전술에서 본인이 쓰러지면? 포로?)
- [ ] 성장/경험치 유무 (프로토타입은 고정 권장)
- [ ] 충성도·등용 시스템 (프로토타입 제외 후보)
- [ ] 초기 등장 장수 명단 (정통 삼국지 vs 가상 — [01 열린 질문](./01-game-overview.md))

## 7.2 병종 (Unit / Troop Types) — 🟡 OPEN

부대를 구성하는 병사 종류. 전략에서는 부대 단위, 전술 확대 시 **개별 병사**로 표현.

### 삼각 상성 (확정 방향)
```
창/보병(槍) ─강→ 기병(騎) ─강→ 궁병(弓) ─강→ 창/보병(槍)
```

| 병종 | 강함 | 약함 | 특징 |
|------|------|------|------|
| 창/보병 (槍) | 기병 | 궁병 | 방어·전선 유지, 저속 |
| 궁병 (弓) | 보병 | 기병 | 원거리 딜, 근접 취약 |
| 기병 (騎) | 궁병 | 창병 | 고기동, 측후방 돌격 |

### 부대 스탯 (초안)
| 스탯 | 의미 |
|------|------|
| 체력(병력 수) | 부대 규모. 0 → 전멸 |
| 공격 | 틱당 가하는 피해 |
| 방어 | 피해 감소 |
| 사기 | 0 접근 시 패주/붕괴 |
| 이동속도 | Hex 이동/전방 전환 속도 |
| 전방(facing) | 전면/측면/후면 판정 기준 |

### 정할 것 (OPEN)
- [ ] 병종을 3종으로 고정 vs 세분화(경기병/중기병, 노병 등)
- [ ] 상성 배율 수치 (예: 상성 우위 시 피해 ×1.5?)
- [ ] 궁병 사거리/탄약, 근접 전환 규칙
- [ ] 병종별 지형 적합성 (기병=평지 강/숲 약 등 → 7.3과 연결)
- [ ] 부대 규모(병력 수) 표준값과 편성 비용 (→ [02 군사](./02-management-layer.md))
- [ ] 측/후면 피격 시 스탯 보정 수치 (→ [03 3.4](./03-strategy-layer.md))

## 7.3 지형 (Terrain) — 🟡 OPEN

Hex 타일 단위. **전략**에서는 수치 보정, **전술**에서는 구체 연출·상호작용으로 확장.

### 지형 타입 (초안)
| 지형 | 전략 효과 (제안) | 전술 연출/상호작용 (제안) |
|------|------------------|---------------------------|
| 평지 | 보정 없음, 기동 자유 | 개활지 |
| 고지/언덕 | 사거리↑·방어↑, 진입 이동↓ | 경사·고지 점령 |
| 숲 | 은폐(피격↓)·이동↓, 사격 제한 | 매복 |
| 강/물가 | 통행 불가 또는 큰 페널티 | 물가·도하 연출 |
| 다리/도하지점 | 병목(소수로 다수 저지) | 도하 중 취약 |
| 협곡/벼랑 | 통행 제한 | **낙석 트리거** 등 발동형 |

### 정할 것 (OPEN)
- [ ] 지형 종류 최종 셋 (프로토타입은 평지+1~2종)
- [ ] 효과 수치(이동 코스트·방어/사거리 보정·사기 영향)
- [ ] **발동형 지형**(낙석·화공) 도입 범위 — 재미 vs 구현비용 (→ [04 4.7 #4](./04-tactics-layer.md))
- [ ] 병종×지형 적합성 매트릭스 (7.2와 교차)
- [ ] 날씨/시간대 등 환경 변수 추가 여부 (프로토타입 제외 후보)
- [ ] 전략 Hex 효과 ↔ 전술 연출의 일관성 규칙
