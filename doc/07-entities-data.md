### 장수 전투 역할 & 생존 — 부대군기 / 대장군기 (1차 확정, 2026-07-01 갱신)

장수는 **버프형 + 일기토 하이브리드**다. 출진 중에는 부대에 버프를 주는 전투 유닛이자,
**HP를 가진 개별 전투 주체**로서 직접 싸운다(일기토 가능).

**부대군기(部隊軍旗)** — 각 유닛 중앙에 두는 **지휘 통제의 중심**. 전술에서는 **'깃발병' 병종으로 표현**된다(잠정, → [7.2](#72-병종-unit--troop-types--🟡-open-능력치-1차-확정-2026-07-01)).
- **이동속도 = 일반 보병의 1/2**(매우 느림). 군기를 전진시켜야 지휘 범위가 따라온다.
- **부서지지 않은 동안**의 역할:
  - **명령**: 지휘 반경(**=통솔 비례**)을 표시하고, 반경 내 유닛에 명령 하달. → [03 3.2.1](./03-strategy-layer.md)
  - **리젠 포인트**: 장수 HP 회복 및 재출진 지점. (HP 소진 → 군기 옆 **휴식** → 리젠 → 재출진)
- **부서지면**: 명령 불가 + 장수 **리젠 불가** + 해당 부대 **사기 저하**.

**도주 & 재편성 (대장군기)** — ⚠️ "해산" 용어 폐기(유닛 소멸로 오해). **도주**로 통일.
- **사기 0 AND 장수 HP 0** → **부대군기 유무와 무관하게 부대는 도주**한다.
- 도주 유닛은 **전략 화면 후방의 대장군기(大將軍旗)** 로 **집결**한다(소멸 아님).
- 대장군기에 **일정 수가 모이고 + 여분의 출진 가능 장수**가 있으면
  → **낮은 사기이지만 다시 출진 가능한 유닛**으로 **재편성**된다.

```
출진(버프+전투) ─HP0→ 부대군기 옆 휴식 ─리젠→ 재출진
부대군기 파괴 → 명령·리젠 불가 + 사기↓
(사기0 & 장수HP0) → 도주 → 대장군기 집결 → [일정 수 + 여분 장수] → 저사기 유닛 재편성
```

> 🟡 OPEN: 부대군기·대장군기 위치/HP/방어/피격 판정, 재편성 임계 수·재편성 시작 사기값, 여분 장수 배정 규칙.

### 장수 상태 & 계략 (1차 확정 2026-06-30)

**상태**
- **출진(出陣)**: HP가 남아 전장에서 싸우는 상태. 부대 버프 제공 + 직접 교전.
- **휴식**: HP 소진 후 **부대군기 옆에서 HP를 회복**하는 상태. 회복 완료 → 재출진(리젠).

**교전 방식 — 성향**
- 출진 중 장수는 **성향에 따라 근접 / 원거리** 교전 상태가 된다. (성향 = 장수별 속성, 추후 데이터화)

**전술계략(計略)** — 명령·계략 3종 중 하나. [04 4.5.1](./04-tactics-layer.md)
- **차징 전에 계략 종류·타겟 위치를 모두 확정**한 뒤 시작한다.
- 발동하려면 **제자리에서 일정 시간 '작전포인트'를 모은다(차징)**.
- 차징 중 **일정 수준 이상의 공격을 받으면 취소**(중단)된다.
- 작전포인트가 가득 차면 발동: **화계 · 낙석 · 매복** 등.
- **범위 = 부대군기 명령반경 + 지력(智)** — 지력이 높을수록 더 멀리/넓게. (통솔→명령반경과 대칭)
- ⚠️ **전략계략과 다르다**(전략계략 = 전략 화면 전용, 경영 부여 — [04 4.5.1](./04-tactics-layer.md)).

```
출진(근접/원거리 교전) ──계략 선택→ 제자리 차징(작전P 축적)
   └ 차징 중 피격 누적 ≥ 임계 → 취소
   └ 작전P 가득 → 계략 발동(범위 = 지휘반경 + 지력)
```

**전술계략 간파 (상대측 지략 주사위)**

차징이 시작되면, 방어측은 **지력 기반 주사위**(운 = 시드 RNG, [05](./05-tech-architecture.md))를 굴려
**무슨 전술계략인지** 간파를 시도한다.

| 주사위 결과 | 획득 정보 |
|------|-----------|
| **대성공** | 계략 **종류 + 타겟 지점** |
| **성공** | 계략 **종류**만 |
| **실패** | 알 수 없음 |

**주사위 보정 — 특기로만 작동** (계략 자체 속성이 아님):
- **간파** 특기(방어측): 주사위에 **+보정** → 간파 성공이 쉬워짐.
- **위보** 특기(차징측): 방어측 주사위에 **−보정** → 간파를 방해.

> ⚠️ **간파·위보는 장수 특기다.** 차징 시 주사위 ±보정으로만 작용한다.
> (거짓정보 노출은 복잡도 때문에 도입하지 않음 — 결정 2026-07-01.)

### 장수 특성(특수기) 후보 (🟡 수집 중 — 상세는 별도 상세화)

> **원칙: 장수 특성은 액티브가 아니라 패시브다.** 유저가 발동하는 게 아니라, **특정 조건이
> 충족되면 자동 발동**한다. 대개 **기본 규칙의 조건/수치를 유리하게 바꾸는** 형태다. (확정 2026-07-01)

확정 골격이 아니라 **아이디어 수집 목록**. 세부 수치·중첩 규칙은 미정.

| 특성 | 발동 조건 | 효과 | 연결 |
|------|-----------|------|------|
| **고무** | 장수 근접전투 지속 | 사기 상승 조건 완화: **10→7초, 최대 2회**(기본 10초·1회) | [03 3.6.1](./03-strategy-layer.md) |
| **간파** | 상대 전술계략 차징 | 간파 주사위 **+보정**(방어측) | 전술계략 간파 |

| **퇴로** | 아군 유닛 도주 | 병사들이 더 빨리 이탈(**도주 손실↓**) | 도주 |

## 7.2 병종 (Unit / Troop Types) — 🟡 OPEN (능력치 1차 확정 2026-07-01)

**한 유닛(부대)은 여러 병종으로 구성**된다(대형 깊이: 앞 방패 → 창 → 궁, 뒤 기병 → [04 4.2.1](./04-tactics-layer.md)).
각 병종은 **장수처럼 자체 능력치**를 가진다.

### 병종 로스터 (5종)

> **병종간 상성 미도입 (확정 2026-07-02).** 창>기>궁>창 식의 **삼각 상성 배율은 쓰지 않는다.**
> 병종 차별화는 **능력치(공격/방어/사거리/저지/이동/회전) + 지형 적합성 + facing(측·후면)** 으로만 표현한다.
> (상성 테이블은 룰 복잡도 대비 실익이 낮아 제외 — 능력치·지형·방향으로 자연스러운 우열이 나오도록 함.)

| 병종 | 특징 |
|------|------|
| 방패병(검병, 盾) | 짧은 사거리, 대열유지·방어용, 낮은 공격·이동 |
| 창병 (槍) | 방패 너머 공격 사거리, **높은 저지력**, 보통 공/방/이동 |
| 궁병 (弓) | 긴 사거리, 낮은 방어, 근접 취약 |
| 기병 (騎) | 높은 이동·강한 공격, **방향 전환 느림**, 저지 대신 **돌파** |
| 깃발병 (旗) | 공격력 없음, 보통 방어. **= 부대군기 유닛**(전술 표현) → [7.1](#장수-전투-역할--생존--부대군기--대장군기-1차-확정-2026-07-01-갱신) |

### 병종 능력치 (1차 확정)
| 능력치 | 의미 |
|--------|------|
| 공격력 | 가하는 피해 |
| 방어력 | 받는 피해 감소 |
| 병수(HP) | 병력 수 = HP. 0 → 소멸 |
| **부상병** | **죽지 않고 복구 가능한 HP.** 도주·회복으로 되살아난다(전사와 구분) |
| 이동력 | 이동 속도 |
| **회전** | **대열(방향) 전환 속도.** 낮으면 측·후면을 내주기 쉬움 → [03 3.4](./03-strategy-layer.md) |
| 사거리 | 공격 도달 거리. 예: **궁병 7 / 창병 2 / 방패병 1** |
| **저지력** | 공격 시 **상대의 이동력을 얼마나 감소**시키는지의 기준 |
| **공속** | **공격 속도.** 기본 공격 횟수(전열 폭 기반)에 곱하는 계수 → [03 3.6.2](./03-strategy-layer.md) |
| **치명율** | 피해 중 **전사(영구)** 비율. 나머지는 **부상(복구)**. → [03 3.6.2](./03-strategy-layer.md) |

> **전사 vs 부상병:** 피해로 병수(HP)가 줄 때 일부는 **전사(영구 손실)**, 일부는 **부상병(복구 가능)**.
> 부상병은 도주·회복(대장군기 재편성·휴식)으로 되돌아온다. → [7.1 도주·재편성](#장수-전투-역할--생존--부대군기--대장군기-1차-확정-2026-07-01-갱신)
>
> **참고:** **사기**는 병종이 아니라 **유닛 단위** 속성이고([03 3.6.1](./03-strategy-layer.md)),
> **facing(방향)** 은 전술에서 **병종 그룹 단위** 연속 각도다([04 4.2.1](./04-tactics-layer.md)).

### 병종별 능력치 등급 (1차 제안 2026-07-01)

병종 능력치는 **수치가 아니라 등급**으로 표기한다(장수는 0~100 수치 — 별도 스케일).

> **등급 스케일:** `S > A > B > C > D` (표준은 **A~D**), 특수 병종/능력에만 **S**(특수 최상)·**E**(없음/무력).

| 병종 | 공격 | 방어 | 이동 | 회전 | 사거리 | 저지 | 공속 | 치명율 | 특수 |
|------|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|------|
| 방패병(검병) | C | A˟ | C | B | D | B | C | C˟ | 대열유지 |
| 창병 | B | B | B | A | C | A | A | B˟ | 방패 너머 공격 |
| 궁병 | B | C | B | A | A | A | B | D˟ | 근접 취약·전열무시 사격 |
| 기병 | A | B | A | D | D˟ | E | **S** | A˟ | 돌파·이동사격 |
| 깃발병 | E | B | D | C | D˟ | D | — | — | = 부대군기(보병 1/2 속도) |

> **저지 확정 (2026-07-02):** 창·궁 = A(최대) / 방패 = B(보통) / 기병 = E(저지 없음, **돌파**) / 깃발 = D.
> **공속 확정 (2026-07-03):** 방패 C / 창 A / 궁 B / 기병 **S**(이동 중 다회 타격). 깃발병은 공격 없음(—).
> **치명율 초안 (2026-07-03):** 기병 A / 창 B / 방패 C / 궁 **D**(부상 위주 마모) / 계략(별도) S~A. → [3.6.2](./03-strategy-layer.md)
> **˟ = 미명시 → 역할 기반 제안(확인 필요).** 남은 ˟: 방패 방어·기병 사거리·깃발 사거리(+치명율 초안값).
> **깃발병 = 부대군기 유닛**(확정 2026-07-02): 이동 D는 부대군기 "보병 1/2"([7.1](#장수-전투-역할--생존--부대군기--대장군기-1차-확정-2026-07-01-갱신))에서 옴.
> **병수(HP)·부상병** 등급은 편성/규모로 별도 결정(미정).

### 정할 것 (OPEN)
- [x] 병종 능력치 셋 → **공격/방어/병수(HP)/부상병/이동력/사거리/저지력**
- [x] **병종 로스터 1차** → 방패병(검병)·창병·궁병·기병·깃발병 (5종)
- [x] **능력치 등급 스케일** → S/A/B/C/D + 특수 E (병종은 등급, 장수는 수치)
- [x] **저지력 등급 확정** → 창·궁 A / 방패 B / 기병 E(돌파) / 깃발 D
- [x] **공속 등급 확정** → 방패 C / 창 A / 궁 B / 기병 S
- [x] **병종간 상성 미도입** → 삼각 상성 배율 제거, 능력치+지형+facing으로 대체
- [x] **깃발병 = 부대군기 유닛** (확정 2026-07-02)
- [x] **전사 : 부상병 분배 규칙** → **치명율**로 분배(피해×치명율=전사). 초안값 확정, 계수 환산 OPEN → [3.6.2](./03-strategy-layer.md)

## 7.3 지형 (Terrain) — 🟡 OPEN

Hex 타일 단위. **전략**에서는 수치 보정, **전술**에서는 구체 연출·상호작용으로 확장.

### 지형 항목 (데이터-레디 초안)
전략은 Hex 타일 1개 = 지형 1종. 전술은 아래 지형들이 **혼합**되어 깔린다(다음 절).

| id | 지형 | 전략 효과 (제안) | 전술 연출/상호작용 (제안) |
|----|------|------------------|---------------------------|
| `plain` | 평지 | 보정 없음, 기동 자유 | 개활지 |
| `stream` | 개천 | 이동 약간↓, 소규모 도하 | 얕은 물·도하(강보다 약함) |
| `river` | 강/물가 | 통행 불가 또는 큰 페널티 | 물가·도하 연출 |
| `hill` | 언덕/고지 | 사거리↑·방어↑, 진입 이동↓ | 경사·고지 점령 |
| `mountain` | 산악 | 통행 크게 제한, 방어↑↑, 소수 저지 | 험지·능선 |
| `forest` | 숲 | 은폐(피격↓)·이동↓, 사격 제한 | 매복 |
| `ford` | 다리/도하지점 | 병목(소수로 다수 저지) | 도하 중 취약 |
| `ravine` | 협곡/벼랑 | 통행 제한 | **낙석 트리거** 등 발동형 |

### 전략→전술 지형 혼합 (확정 2026-07-01)
전략 화면의 한 타일이 '숲'이어도 전술 화면 **전체가 숲일 수는 없다.**
- **대표 지형(전략 타일의 지형)이 전술 맵의 ≥ 60%** 를 차지.
- **나머지(~40%)** 는 평지·개천·언덕·산악 등 **다른 지형이 섞여** 들어간다.
- 목적: 전술 전장에 **지형 다양성** → 대형·기동·매복의 국지 변수를 만든다.

> 혼합 생성은 **시드 기반 절차적**으로 → 같은 전투는 항상 같은 지형. ([05 결정성](./05-tech-architecture.md))

### 병종 × 지형 적합성 (1차 확정 2026-07-02)

각 병종이 지형에서 얼마나 유리한가. **◎ 유리 / ○ 보통 / △ 불리 / ✕ 통행·전투 곤란.**
(전투 효율 + 이동을 함께 반영한 개략치. 수치 보정은 [7.3 효과 수치 OPEN])

| 지형 | 방패병 | 창병 | 궁병 | 기병 | 깃발병 |
|------|:---:|:---:|:---:|:---:|:---:|
| 평지 | ○ | ○ | ○ | ◎ | ○ |
| 개천 | ○ | ○ | ○ | △ | △ |
| 강 | ✕ | ✕ | ✕ | ✕ | ✕ |
| 언덕/고지 | ○ | ○ | ◎ | △ | △ |
| 산악 | △ | △ | ○ | ✕ | △ |
| 숲 | ○ | ○ | △ | △ | △ |
| 다리/도하 | ◎ | ◎ | ○ | △ | △ |
| 협곡/벼랑 | △ | △ | ○ | ✕ | △ |

> 논리: 기병=평지◎·험지✕, 궁병=고지/산악◎(사거리·시야), 방패·창=병목(다리)◎·전선 유지,
> 깃발병(=부대군기, 저속)은 험지 전반 △. **표 자체는 확정, 수치 환산은 [7.3 효과 수치 OPEN].**
