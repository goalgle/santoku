## 7.1 장수 (Characters / Generals) — 🟡 OPEN (능력치 세트 1차 확정 2026-06-30)

삼국지의 핵심 재미. 부대를 지휘하거나 도시를 다스린다. 전술 확대 시 **본인이 등장**한다.

### 능력치 6종 (1차 확정 — 0~100 스케일, 코에이류 확장)
| 능력치 | 영향 (제안) | 주 레이어 | 프로토타입 |
|--------|-------------|-----------|------------|
| 통솔 (統) | 부대 사기·방어·행군 결속·협공 저항·**부대군기 지휘 반경** | 전략·전술 | ✅ 작동 |
| 무력 (武) | 근접 전투력·돌격·일기토 | 전략·전술 | ✅ 작동 |
| 지력 (智) | 계략·매복·탐색·간파·함정·**계략 범위** | 전략·전술 | ✅ 작동 |
| 정치 (政) | 내정 생산·건설·민심 | 경영 | ⏸ 데이터만 |
| 매력 (魅) | 등용·외교·민심·이간 저항 | 경영·외교 | ⏸ 데이터만 |
| 운 (運) | 판정 변동폭(치명·회피·발견 등) | 전 레이어 | ✅ 작동(시드) |

> 추가 속성(속도·연령·신분 등)은 **시나리오 흐름에 따라** 뒤에서 더할 수 있다. 세트는 열려 있다.

### 능력치 × 이벤트 영향 (1차안)
| 이벤트 | 주(主) | 부(副) | 레이어 |
|--------|--------|--------|--------|
| **이동**(행군) | 통솔 (결속·행군속도·낙오↓) | 지력 (경로) | 전략 |
| **탐색**(정찰·수색) | 지력 (정보 정확도) | 운 (발견율) | 전략 |
| **경계**(척후·간파) | 지력 (매복 간파) | 통솔 (경계망 유지) | 전략 |
| **계략**(화공·유인·이간) | 지력 | 매력 (이간·민심 흔들기) | 전략·경영 |
| **매복**(설치·기습) | 지력 | 통솔 (은밀 기동) | 전략·전술 |
| **전투** | 무력(돌격·일기토) + 통솔(사기·방어) | 운 (치명·회피) | 전략·전술 |
| **내정/건설** | 정치 | 매력 (민심) | 경영 |
| **등용/외교** | 매력 | 정치 | 경영·외교 |

### 운(運) 처리 — 시드 기반 RNG (확정 2026-06-30)
- 운은 무작위 판정의 **확률 가중치**로만 작동(치명·회피, 탐색 발견, 계략 성공 등의 ±변동).
- **전역 시드 고정** → 같은 시드·입력이면 100% 재현. 리플레이·디버깅·밸런싱 보존.
- doc 05 "순수 데이터 + 결정적 시뮬" 원칙과 양립: RNG는 **시드 의존 결정함수**로 취급.

### 프로토타입 범위 (확정 2026-06-30)
- **작동**: 통솔·무력·지력·운 (전투·필드 이벤트 중심)
- **데이터만(미사용)**: 정치·매력 — 경영/외교 레이어 구현 시 활성화. 칸은 미리 둬 데이터 일관성 유지.
- 등용·충성도는 프로토타입 제외 후보(CLAUDE.md)와 일관 — 매력 효과는 그 시점에 함께 켠다.

### 장수 전투 역할 & 생존 — 부대군기 / 대장군기 (1차 확정 2026-06-30)

장수는 **버프형 + 일기토 하이브리드**다. 출전 중에는 부대에 버프를 주는 전투 유닛이자,
**HP를 가진 개별 전투 주체**로서 직접 싸운다(일기토 가능).

- 각 유닛(부대)은 중앙에 **부대군기(部隊軍旗)** 유닛을 둔다. 이 군기는 **지휘 통제의 중심**이다:
  - **이동속도 = 일반 보병의 1/2**(매우 느림). 군기를 전진시켜야 지휘 범위가 따라온다.
  - **지휘 반경**을 가져, 액티브 포즈 명령은 **반경 내 유닛에게만** 하달된다. **반경 = 장수 통솔(統)에 비례**. → [03 3.2.1](./03-strategy-layer.md)
- **장수 HP 소진** → 부대군기로 복귀해 휴식. **부대군기가 존재하는 한 계속 리젠**(복귀→재출전 반복).
- **부대군기 파괴** → 장수 리젠 불가 + 해당 부대 **사기 저하**.
- **사기 바닥 + 장수 부재** → 해당 유닛 잔존 병력 전원 **해산**.
- 해산 병력은 **전략 화면 후방의 대장군기(大將軍旗)** 로 모인다(재편성 자원).

```
출전(버프+전투) ─HP 0→ 부대군기 복귀(휴식) ─리젠→ 재출전
부대군기 파괴 → 리젠 불가 + 사기↓ ─(사기 바닥 & 장수 부재)→ 잔존병 해산 → 대장군기로 집결
```

> 🟡 **부대군기·대장군기는 추후 상세화(OPEN).** 위치/HP/이동·방어/피격 판정, 대장군기 재편성 규칙 등 미정.

### 장수 상태 & 계략 (1차 확정 2026-06-30)

**상태**
- **출진(出陣)**: HP가 남아 전장에서 싸우는 상태. 부대 버프 제공 + 직접 교전.
- **휴식**: HP 소진 후 **부대군기 옆에서 HP를 회복**하는 상태. 회복 완료 → 재출진(리젠).

**교전 방식 — 성향**
- 출진 중 장수는 **성향에 따라 근접 / 원거리** 교전 상태가 된다. (성향 = 장수별 속성, 추후 데이터화)

**계략(計略)**
- 발동하려면 **제자리에서 일정 시간 '작전포인트'를 모은다(차징)**.
- 차징 중 **일정 수준 이상의 공격을 받으면 취소**(중단)된다.
- 작전포인트가 가득 차면 발동: **화계 · 낙석 · 매복** 등.
- **계략 범위 = 기본 지휘 반경 + 지력(智)** — 지력이 높을수록 더 멀리/넓게. (통솔→지휘반경과 대칭)

```
출진(근접/원거리 교전) ──계략 선택→ 제자리 차징(작전P 축적)
   └ 차징 중 피격 누적 ≥ 임계 → 취소
   └ 작전P 가득 → 계략 발동(범위 = 지휘반경 + 지력)
```

**계략 간파 (상대측 지력·운 판정)**

상대 장수가 작전포인트를 모으는 동안, 방어측은 **지력 + 운 주사위**를 굴려 간파를 시도한다.
(운 = 시드 RNG → 재현 가능, [05](./05-tech-architecture.md))

| 판정 | 획득 정보 |
|------|-----------|
| **대성공** | 계략 **종류 + 타겟 지점** |
| **성공** | 계략 **종류**만 |
| **실패** | 알 수 없음 |
| **대실패 + 시전자 '위보' 보유** | **잘못된(거짓) 계략 정보** 획득 |

> **위보(僞報)** = 장수 특수기/특성 후보. 적 간파의 대실패를 역이용해 **허위 정보**를 흘린다
> (역정보전). → 특수기 체계와 연결(아래 OPEN).

## 7.2 병종 (Unit / Troop Types) — 🟡 OPEN

부대를 구성하는 병사 종류. 전략에서는 부대 단위, 전술 확대 시 **개별 병사**로 표현.

### 삼각 상성 (확정 방향)
```
창/보병(槍) ─강→ 기병(騎) ─강→ 궁병(弓) ─강→ 창/보병(槍)
```

| 병종 | 강함 | 약함 | 특징 |
|------|------|------|------|
| 창/보병 (槍) | 기병 | 궁병 | 방어·전선 유지, 저속 |
| 궁병 (弓) | 보병 | 기병 | 원거리 딜, 근접 취약 |
| 기병 (騎) | 궁병 | 창병 | 고기동, 측후방 돌격 |

### 부대 스탯 (초안)
| 스탯 | 의미 |
|------|------|
| 체력(병력 수) | 부대 규모. 0 → 전멸 |
| 공격 | 틱당 가하는 피해 |
| 방어 | 피해 감소 |
| 사기 | 0 접근 시 패주/붕괴 |
| 이동속도 | Hex 이동/전방 전환 속도 |
| 전방(facing) | 전면/측면/후면 판정 기준 |

### 정할 것 (OPEN)
- [ ] 병종을 3종으로 고정 vs 세분화(경기병/중기병, 노병 등)
- [ ] 상성 배율 수치 (예: 상성 우위 시 피해 ×1.5?)
- [ ] 궁병 사거리/탄약, 근접 전환 규칙
- [ ] 병종별 지형 적합성 (기병=평지 강/숲 약 등 → 7.3과 연결)
- [ ] 부대 규모(병력 수) 표준값과 편성 비용 (→ [02 군사](./02-management-layer.md))
- [ ] 측/후면 피격 시 스탯 보정 수치 (→ [03 3.4](./03-strategy-layer.md))

## 7.3 지형 (Terrain) — 🟡 OPEN

Hex 타일 단위. **전략**에서는 수치 보정, **전술**에서는 구체 연출·상호작용으로 확장.

### 지형 타입 (초안)
| 지형 | 전략 효과 (제안) | 전술 연출/상호작용 (제안) |
|------|------------------|---------------------------|
| 평지 | 보정 없음, 기동 자유 | 개활지 |
| 고지/언덕 | 사거리↑·방어↑, 진입 이동↓ | 경사·고지 점령 |
| 숲 | 은폐(피격↓)·이동↓, 사격 제한 | 매복 |
| 강/물가 | 통행 불가 또는 큰 페널티 | 물가·도하 연출 |
| 다리/도하지점 | 병목(소수로 다수 저지) | 도하 중 취약 |
| 협곡/벼랑 | 통행 제한 | **낙석 트리거** 등 발동형 |

### 정할 것 (OPEN)
- [ ] 지형 종류 최종 셋 (프로토타입은 평지+1~2종)
- [ ] 효과 수치(이동 코스트·방어/사거리 보정·사기 영향)
- [ ] **발동형 지형**(낙석·화공) 도입 범위 — 재미 vs 구현비용 (→ [04 4.7 #4](./04-tactics-layer.md))
- [ ] 병종×지형 적합성 매트릭스 (7.2와 교차)
- [ ] 날씨/시간대 등 환경 변수 추가 여부 (프로토타입 제외 후보)
- [ ] 전략 Hex 효과 ↔ 전술 연출의 일관성 규칙
