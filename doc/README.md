# 프로젝트 기획서 (가제: Ancient Average)

삼국지 시대를 배경으로 한 **토탈워류 전략 게임**의 기획 문서 모음입니다.
**경영(턴제) → 전략(실시간) → 전술(확대 실시간)** 의 3개 레이어를 **연속 줌**으로 잇고,
**액티브 포즈**(유저 개입 시 전체 정지)를 핵심 인터랙션으로 삼습니다.

## 한눈에 보기

| 항목 | 내용 |
|------|------|
| 장르 | 전략 시뮬레이션 + 실시간 전술 (토탈워류) |
| 배경 | 삼국지 (고대 중국, 후한~삼국 시대) |
| 핵심 메커니즘 | ① 연속 줌(전략↔전술) ② 액티브 포즈(개입 시 전체 정지) |
| 플랫폼 | 웹 기반 (모바일 우선 → PC 확장) |
| 개발 규모 | 1인 / 소규모 프로토타입 |

## 3개 레이어

| 레이어 | 시간 | 화면 | 역할 |
|--------|------|------|------|
| **경영** | 턴제 | 전체 지도 | 내정·외교·부대 편성·이동 |
| **전략** | 실시간 (빠른 시계) | Hex 전장, 유닛들이 보임 | 부대 단위 지휘 + **교전 자동 계산** |
| **전술** | 실시간 (느린 시계) | 특정 교전 안으로 확대 | 장수·병종·지형 미시 지휘 (자동 계산에 개입) |

> 두 실시간 시계는 **비율(`TIME_RATIO`)로 분리** — 전략은 빠르게, 전술은 느리게. 전술에 몰입한 사이
> 벌어진 전략 상황은 **알림으로 공유**된다(주의의 안개). 액티브 포즈는 양쪽을 동시에 멈춘다.
